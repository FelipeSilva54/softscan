Política de Privacidade do SoftScan

Última atualização: 7 de julho de 2026

Resumo: o SoftScan não tem login, não tem servidor próprio e não envia seus dados para nenhum lugar. Tudo o que o app lê ou guarda fica somente no seu aparelho.

1. Quem somos

O SoftScan é um aplicativo gratuito para Android que lê QR Code Pix e código de barras de boleto bancário, exibe os dados extraídos (valor, vencimento, beneficiário, etc.) e permite copiar ou compartilhar essas informações. Esta política explica como o app trata dados ao ser usado.

2. Quais dados o app acessa

  • Câmera: usada exclusivamente para ler o QR Code ou código de barras na tela do scanner, em tempo real. O SoftScan não grava, não salva e não transmite imagens ou vídeo da câmera para nenhum servidor. Nenhuma foto é armazenada.
  • Conteúdo do código escaneado: os dados do Pix ou boleto (valor, vencimento, beneficiário, linha digitável, etc.) são processados e validados (checksum módulo 10/11 para boleto, CRC16 para Pix) diretamente no aparelho.
  • Histórico de leituras: se você optar por salvar uma leitura, ela é guardada apenas no armazenamento local do próprio dispositivo. Esse histórico nunca sai do aparelho e pode ser apagado a qualquer momento dentro do app ou desinstalando o aplicativo.
  • Área de transferência (clipboard): ao tocar em "copiar", o código lido é copiado para a área de transferência do sistema, para que você possa colar em outro app (ex.: seu banco).

3. O que o app NÃO faz

  • Não exige login, cadastro ou conta de usuário.
  • Não possui servidor ou backend próprio: nenhum dado escaneado é enviado pela internet.
  • Não contém anúncios.
  • Não usa SDKs de analytics, rastreamento ou publicidade de terceiros.
  • Não coleta, vende ou compartilha dados pessoais com terceiros.
  • Não acessa contatos, localização, arquivos ou outros dados do aparelho além do necessário para a câmera e para salvar o histórico localmente.

4. Compartilhamento (recurso "Compartilhar")

Ao tocar em "compartilhar", o SoftScan abre o menu de compartilhamento nativo do Android. A escolha de para onde enviar o conteúdo (WhatsApp, e-mail, etc.) é feita por você; o SoftScan não tem acesso ao que acontece depois que o conteúdo sai desse menu e não recebe nenhuma confirmação ou dado de volta.

5. Tela de doação (Pix)

O app exibe uma chave Pix estática para quem desejar apoiar o projeto voluntariamente. O SoftScan apenas copia essa chave para a área de transferência: o pagamento em si é feito pelo seu próprio aplicativo bancário, fora do SoftScan. O app não coleta, processa ou tem acesso a nenhum dado sobre eventuais doações (valor, data, dados bancários do doador, etc.).

6. Armazenamento e segurança

Todos os dados de uso do app (histórico de leituras) ficam salvos localmente no armazenamento do dispositivo, protegidos pelas mesmas proteções do sistema operacional Android para dados de aplicativos. Ao desinstalar o SoftScan, esses dados são removidos junto com o app.

7. Crianças

O SoftScan não é direcionado a crianças e não coleta intencionalmente dados de menores de idade. Como o app não coleta dados pessoais de ninguém, isso vale igualmente para usuários de qualquer idade.

8. Alterações nesta política

Esta política pode ser atualizada caso o app ganhe novas funcionalidades que alterem o tratamento de dados. A versão mais recente estará sempre disponível nesta mesma página, com a data de atualização indicada no topo.
